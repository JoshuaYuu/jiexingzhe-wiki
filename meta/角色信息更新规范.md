> 本文档规范角色信息的创建、更新和维护流程，确保Wiki中角色信息的一致性和完整性。

| 角色 | 页面 | 当前状态 |
|------|------|----------|

## 二、角色信息更新检查清单

### 创建新角色时必须检查

#### 1. 独立角色页面（S级/A级）
- [ ] 创建 `characters/角色名.html`
- [ ] 使用标准模板（参考林夕/默页面）
- [ ] 包含Canon状态标记
- [ ] 添加图片保护脚本
- [ ] 响应式设计适配移动端

#### 2. 角色档案备份（Markdown）
- [ ] 创建 `characters/角色名_档案.md`
- [ ] 包含完整人设信息
- [ ] 作为页面内容的备份和参考

#### 3. 角色列表页面更新
- [ ] 更新 `characters.html`
- [ ] 缩略图使用 `<a href="characters/角色名.html">` 链接
- [ ] 更新角色数据对象（openModal用）
- [ ] 确保职业、元素、势力标签正确

#### 4. 搜索功能更新
- [ ] 更新 `search.html` 中的搜索数据
- [ ] 添加角色关键词
- [ ] 确保链接指向正确

#### 5. 相关页面引用更新
- [ ] 检查 `story.html` 剧情大纲中的引用
- [ ] 检查 `factions.html` 势力页面中的引用
- [ ] 检查 `meta/timeline.html` 时间线中的引用
- [ ] 更新其他角色页面的关系描述

#### 6. 交叉链接检查
- [ ] 确保角色页面链接到相关角色
- [ ] 确保角色页面链接到相关势力
- [ ] 确保角色页面链接到相关剧情章节

## 四、信息一致性检查点

### 基础信息一致性
| 检查项 | 检查位置 | 说明 |
|--------|----------|------|
| 角色名称 | 所有页面 | 确保统一使用正式名称或代号 |
| 年龄 | 角色页、剧情页 | 随剧情时间线变化需同步更新 |
| 职业/道 | 角色页、系统页 | 与六道系统定义保持一致 |
| 元素属性 | 角色页、系统页 | 与九大元素定义保持一致 |
| 所属势力 | 角色页、势力页 | 与势力设定保持一致 |

### 关系描述一致性
| 关系对 | A角色页面描述 | B角色页面描述 | 检查点 |
|--------|---------------|---------------|--------|
| 林夕-默 | 师姐-师弟 | 师姐-师弟 | 描述一致 |
| 林夕-关锋 | 队长-副队长 | 队长-副队长 | 描述一致 |
| 默-关锋 | 战友 | 导师-学生 | 视角差异合理 |

### 剧情时间线一致性
| 检查项 | 检查位置 | 说明 |
|--------|----------|------|
| 觉醒时间 | 角色页、剧情页、时间线页 | 确保时间一致 |
| 关键事件 | 角色页、剧情页 | 确保描述一致 |
| 能力获得 | 角色页、剧情页 | 确保章节对应正确 |

## 六、常见问题处理

### Q1：角色名称变更如何处理？
**示例**：陈默 → 默

1. 更新角色独立页面标题和内容
2. 更新角色列表页面显示名称
3. 全局搜索替换所有引用（保留本名作为别名）
4. 更新搜索数据关键词
5. 在角色页面添加"本名"字段说明

### Q2：角色等级提升如何处理？
**示例**：B级角色 → A级角色（需要独立页面）

1. 创建独立HTML页面
2. 将 `onclick="openModal()"` 改为 `<a href="">`
3. 创建Markdown档案备份
4. 更新所有相关引用链接

### Q3：剧情设定冲突如何处理？
1. 记录冲突点
2. 评估影响范围
3. 与用户确认最终设定
4. 统一更新所有相关页面

## 七、文件命名规范

### HTML页面
- `characters/角色名.html`（小写，无空格）
- 示例：`linxi.html`, `chenmo.html`

### Markdown档案
- `characters/角色名_档案.md` 或 `characters/角色名_profile.md`
- 示例：`林夕_完整角色人设档案.md`

### 图片资源
- `角色名/avatar.png`（头像）
- `角色名/CG_场景名.png`（剧情CG）
- `角色名/立绘_描述.png`（立绘）

## 八、Canon状态使用规范

| 状态 | 含义 | 使用场景 |
|------|------|----------|
| **已确定** ✅ | 设定已最终确认 | 核心设定、已发布剧情 |
| **提议中** 💡 | 设定正在讨论 | 新角色、待确认剧情 |
| **推测** ❓ | 基于线索的推测 | 隐藏设定、玩家推论 |
| **已废弃** ❌ | 设定已废弃 | 旧版本、已修改设定 |
