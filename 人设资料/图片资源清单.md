## 图片命名规范

### 格式
`角色名_类型_描述.扩展名`

### 示例
- `林夕_avatar.png` - 头像
- `林夕_立绘_默认状态.png` - 立绘
- `林夕_CG_月下疗伤.png` - 事件CG
- `默_立绘_观道状态.png` - 立绘
- `关锋_立绘_优雅冷面.png` - 立绘

### 类型标识
| 标识 | 含义 |
|------|------|
| `avatar` | 头像（用于Wiki角色卡片） |
| `立绘` | 全身立绘 |
| `CG` | 事件CG/剧情插图 |
| `概念` | 概念设计图 |
| `武器` | 武器设定图 |
| `三视图` | 角色三视图 |

## Wiki图片引用路径

### 角色头像
```
characters/角色名/avatar.png
```

### 角色立绘
```
characters/角色名/立绘_状态.png
```

### 事件CG
```
characters/角色名/CG_事件名.png
```

### 示例
```html
<img src="characters/linxi/avatar.png" alt="林夕头像">
```
